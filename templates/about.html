About

Othello CharKey Extractor (Othello Character Keywords Extractor) is a web app project for LTCS180. The web app can be used to extract keywords from dialogues of significant characters in Shakespeare's play, Othello. The identification and extraction of keywords is performed by KeyBert.
The character, the number of keywords to extract, and if desired, a specific act and/or scene, can be selected to determine whose and which dialogues to extract a number of keywords from. The extracted keywords can then be examined and used to help identify the role the selected character plays in a selected act/scene of Othello or in Othello in general.

Project Resources:

  • Othello Text: https://dracor.org/shake/othello#downloads
  • KeyBert: https://maartengr.github.io/KeyBERT/
  • Early Modern English (as Shakespeare is classified "Early Modern English") and Middle English (in case of some overlap) stopwords found online were used with KeyBert.
    • Early Modern English Stopwords: http://earlymodernconversions.com/wp-content/uploads/2013/12/stopwords.txt
    • Middle English: https://docs.cltk.org/en/latest/cltk.stops.html
